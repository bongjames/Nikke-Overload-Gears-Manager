Welcome to Overload Gear Manager 👋

Track your gear rolls, get reroll advice, and spend rocks efficiently.
1
Add your Nikkes — Use the "+ Add Nikke" button in the Nikkes tab to add the characters you're gearing up.
2
Set line priorities — Each Nikke gets default priorities (ATK + Elemental). Tweak these in the Line Priorities tab if needed.
3
Record your gear rolls — Select a Nikke, pick a gear slot, and enter the stat lines you rolled in-game.
4
Get a verdict — The app tells you whether to keep, lock + reroll, or reset — with estimated rock costs.
5
Check the Overview — Once you have a few Nikkes entered, the Overview ranks which slot to reroll next for the best return on rocks.
You can re-open this guide from the top bar any time.
